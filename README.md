# 🧾 인증 서비스가 들어간 개인 지출 관리 애플리케이션 (React) 🧾

## 구현 사항
- [x] 지출 관리 시스템에 회원가입 / 로그인 기능이 존재합니다.
- [x] 로그인 후 글 작성, 수정 및 삭제가 가능합니다.
- [x] json-server 를 이용해 지출 데이터에 대한 CRUD 를 기능하게 했습니다.
- [x] Tanstack Query (ReactQuery) 로 응답합니다.

- [x] 작성 유저만 수정 및 삭제가 가능합니다.
